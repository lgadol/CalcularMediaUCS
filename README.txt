------------------------------------- TUTORIAL FOR EXECUTABLE ------------------------------------------





To generate the .jar executable, enter the path "CalcularMediaUCS\CalcularMediaUCS\src" and execute the following command "javaccalculalmediaucs/CalcularMediaUCS.java".

After that, in the same path, create a text file, with the name "manifest" and inside it type "Main-Class:calculatemediaucs.CalcularMediaUCS".

After that, in "CalcularMediaUCS\CalcularMediaUCS\src" run the following command "jar cvfm CalcularMediaUCS.jar manifest.txtcalculmediaucs/*.class".
